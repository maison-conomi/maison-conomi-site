# Maison Conomi サイト概要

## このサイトについて

Maison Conomi の作品やプロフィールを紹介するための、HTML / CSS / JavaScript で作られたシンプルな静的サイト。

トップページ、About、Works、Gallery、ほかほかパンダ の5ページ構成で、画像素材は `images/`、見た目の調整は `css/` にまとまっている。

公開で使用する画像は基本的に軽量化した `*-site.webp` を使い、変換前の元画像はローカル保管用の `original-images/` に退避する。`original-images/` はGitHub Pages公開用にはアップロードしない。

共通ナビには `about`、`works`、`gallery`、ほかほかパンダのアイコン、外部リンクの `shop` と `links` がある。

## ファイル構成

- `index.html`
    - トップページ
    - メインビジュアル画像 `images/top-site.webp` を表示している
    - トップ画像はヘッダー下の表示領域いっぱいに、中央揃えで表示している
    - `images/top-hokahokapanda-space-cosmo-crop-site.webp` をTOP画像の上に重ね、マウスやタップでつまんで投げられるようにしている
    - 投げたあとは慣性で動き、TOP枠の端に当たると跳ね返りながら回転する
- `about.html`
    - ヘッダー画像なしでプロフィールから始まる、What I make、Favorite things、History、SNS / Contact のページ
    - `conomi` のプロフィールでは、アトリエでの制作と `めんだこぬ` としてのネット活動を紹介している
    - プロフィール内から `hokapan.html` へ遷移できる、ほかほかパンダ紹介リンクがある
    - `めんだこぬ` は X や YouTube で使用する活動キャラクターとしてプロフィール内に表示している
    - `What I make` ではイラスト、絵画、世界観、キャラクター、旅本、絵本、グッズ、ぬいぐるみ、動画、アニメーションを紹介している
    - `Favorite things` では散歩、旅、どうぶつ、植物など、制作の周辺にある好きなものを紹介している
    - `History` では担当作品、自主制作本・絵本、展示やイベント出店を掲載している
    - プロフィール画像は `images/about-conomi-site.webp`、活動キャラクター画像は `images/about-mendaconu.png` を使用している
    - ほかほかパンダ紹介画像やナビアイコンは `images/about-hokahokapanda-site.webp` を使用している
    - About用の見た目は `css/about.css`
- `works.html`
    - お仕事やオリジナル作品を、大きな画像と短い情報の1列一覧で紹介するページ
    - Aboutページに合わせた淡い背景と見出しで、作品画像をゆっくり見られるようにしている
    - フィルター周りは背景や枠を付けず、ページ背景になじませた軽い見た目にしている
    - `Category`、`Type`、`Year` のボタンで作品を絞り込める
    - スクロール後に右下へ上へ戻るボタンを表示する
    - 作品情報は `js/works-data.js` の `worksData` 配列で管理し、一覧表示と年フィルタを自動生成している
    - 仕事・コラボ作品の画像は `images/works/client/`、自主制作の画像は `images/works/original/` にまとめている
    - Worksで表示する作品画像は基本的に `*-site.webp` の公開用軽量版を使っている
    - Works用の見た目は `css/works.css`
- `js/works-data.js`
    - Worksに掲載する作品情報と絞り込み項目をまとめている
    - 作品追加時に基本的に編集するファイル
- `js/works.js`
    - Worksの一覧表示と絞り込み操作を動かす処理
    - 通常の作品追加では編集しない
- `gallery.html`
    - イラスト一覧のページ
    - 画像をクリックすると拡大表示される
    - スマホ版でも横長画像は1行幅で大きく表示する
    - スクロール後に右下へ上へ戻るボタンを表示し、画像モーダル中は非表示にする
    - Galleryで表示する作品画像は基本的に `*-site.webp` の公開用軽量版を使っている
    - Gallery用の見た目は `css/gallery.css`
- `hokapan.html`
    - ほかほかパンダの部屋をイメージしたキャラクターページ
    - ナビではパンダのアイコンで表示している
    - タイトル画像、固定素材、ランダム表示のほかほかパンダ、プロフィールを1画面に配置している
    - PC版は `.hokapan-stage` を、絵と文字を含む有効範囲として扱っている
    - PC版の表示範囲は床の線を基準に詰め、外側の白い余白を含めにくくしている
    - `images/hokahokapanda/fixed/` の床の線、こうみんかん、ドア、窓、マップ、おみせ、つみきを固定配置している
    - 固定素材やランダム表示素材は、軽くなるものだけ `*-site.webp` を使用し、PNGの方が軽い床の線や窓などはPNGのまま使っている
    - `images/hokahokapanda/random/` の画像から、ページ更新ごとにほかほかパンダを1枚ランダム表示している
    - 通常のランダムほかほかパンダはCSSアニメーションでゆっくり部屋の中を動く
    - `hokapan-character-01-site.webp` は静止表示で、ページ更新ごとに素材やプロフィールに重なりにくい白い余白へランダム表示する
    - `hokapan-character-02-site.webp`、`03`、`04`、`05`、`09` は床の線より下を歩き回る
    - 床を歩くほかほかパンダは、画像の中心が床線より上に行かず、ほかほかマートにかぶらない範囲でランダムに移動する
    - `hokapan-character-02-site.webp` は進行方向に合わせて左右反転する
    - 床を歩くほかほかパンダは、左右にゆっくり傾く重めの歩き方にしている
    - `hokapan-character-04-site.webp` はスマホ版で少し大きめに表示する
    - `hokapan-character-06-site.webp` はロケットとして、つかんで投げると進行方向に機首を向けながら部屋の中を飛ぶ
    - `hokapan-character-07-site.webp` はその場でぴょんぴょん跳ねる
    - こうみんかん、ドア、マップはクリックすると説明パネルを表示する
    - おみせは左右の斜線で囲んだ `SHOP!` 表示を付け、外部shopへ遷移する
    - スマホ版ではクリック可能な部屋素材をページ表示時に一度だけふわっと薄くして、タップできる場所を示している
    - 画面上部に、再訪すると違うほかぱんに会えることを伝える青い短いメッセージを表示している
    - スマホ版では再訪メッセージを上部に置き、名前ロゴは少し下げて別グループに見えるようにしている
    - スマホ版では床の線を基準に、ドア、こうみんかん、プロフィールの位置を調整している
    - PC版とスマホ版のプロフィールは中央寄せで、スマホ版は画面サイズに合わせて文字サイズが変わる
    - ほかほかパンダ用の見た目は `css/hokapan.css`
- `css/reset.css`
    - ブラウザごとの余白や表示差をならすためのCSS
- `css/style.css`
    - 全ページ共通のCSS
    - ヘッダー、ナビ、トップページの見た目など
    - WorksとGalleryで使う上へ戻るボタン `.back-to-top` の見た目を管理している
    - トップページだけ左右余白をなくし、`.top-visual` を `object-fit: cover` と `object-position: center` で画面いっぱいに中央トリミング表示
    - `.top-floating-panda` でTOPのほかほかパンダのサイズ、重なり、ドラッグ中の見た目を調整
    - TOPのほかほかパンダのサイズは画面幅に連動しない固定サイズ
- `images/`
    - サイトで使う公開用画像素材の置き場
    - 基本的に軽量化済みの `*-site.webp` を使用する
    - WebPよりPNGの方が軽い小さな透明素材はPNGのまま置く
- `js/`
    - Worksなどの表示データや動きをまとめるJavaScriptファイル置き場
    - `image-protection.js` で画像の右クリック保存やドラッグ保存を抑制している
    - `back-to-top.js` でWorksとGalleryの上へ戻るボタンを制御している
- `original-images/`
    - WebP変換前の元画像をローカルで保管するためのフォルダ
    - GitHub Pages公開用にはアップロードしない
- `markdown/`
    - メモや素材リストなどのMarkdownファイル置き場

## 触るときの目安

- 文章を変えたい
    - 各HTMLファイルの `<p>`、`<h1>`、`<h2>`、`<h3>` の中身を編集する
- メニューを変えたい
    - 各HTMLファイル内の `<ul class="nav-list">` を編集する
- 画像を変えたい
    - `images/` に画像を入れる
    - HTMLの `<img src="images/画像名">` を変更する
    - Worksの素材は `images/works/` にまとめ、仕事・コラボは `client/`、自主制作は `original/` に入れる
    - ほかほかパンダページの素材は `images/hokahokapanda/` にまとめる
    - 公開用に使う大きな画像はWebPへ変換し、`*-site.webp` の名前で参照する
    - 変換前の元画像は `original-images/` に退避し、公開用にはアップロードしない
- トップ画像の見え方を変えたい
    - 公開表示は `images/top-site.webp` を使用している
    - 元画像を差し替えたら、WebP変換した公開用画像も差し替える
    - 枠の比率やトリミング位置は `css/style.css` の `.frame` と `.top-visual` を変更する
    - 上に重なるほかほかパンダの位置や大きさは `css/style.css` の `.top-floating-panda` を変更する
    - ほかほかパンダの投げる動きや跳ね返りは `index.html` 下部の `<script>` を変更する
- 色や余白を変えたい
    - 共通部分は `css/style.css`
    - Aboutは `css/about.css`
    - Worksは `css/works.css`
    - Galleryは `css/gallery.css`
    - ほかほかパンダは `css/hokapan.css`
- ほかほかパンダページの部屋配置を変えたい
    - PC版の外枠比率は `css/hokapan.css` の `.hokapan-room`、中身の座標は `.hokapan-stage` 内の各クラスを調整する
    - `css/hokapan.css` の `.community-button`、`.door-button`、`.window-object`、`.map-button`、`.shop-button`、`.blocks-object`、`.random-hokapan` を調整する
    - 名前ロゴは `.hokapan-name-image` で位置、幅、最大幅を調整する
    - プロフィール文字は `.hokapan-profile-list` で位置、幅、文字サイズを調整する
    - 再訪メッセージは `.hokapan-return-message` で位置、色、文字サイズを調整する
    - PCとスマホで位置が違うため、通常指定と `@media (max-width: 720px)` の両方を確認する
    - スマホ版の床線位置は `--mobile-floor-y` を基準にしている
    - スマホ版の床線の太さは `.floor-line` の `scaleY()` で調整する
    - スマホ版でドアを中央に置く場合は `.door-button` の `left: 50%` と `transform: translate(-50%, -100%)` を使う
    - スマホ版の名前ロゴ、プロフィール文字、再訪メッセージは `@media (max-width: 720px)` 内で個別に調整している
    - スマホ版のプロフィール文字と再訪メッセージは `clamp()` で画面に合わせて拡大縮小している
- ほかほかパンダページのランダム画像を増やしたい
    - `images/hokahokapanda/random/` に画像を追加する
    - `hokapan.html` 下部の `hokapanImages` 配列にファイル名を追加する
    - 公開用は軽量化した `*-site.webp` を追加する
    - 床上を歩かせたい画像は `hokapan.html` 下部の `floorWalkHokapanIds` に番号を追加する
- ほかほかパンダページのクリック先を変えたい
    - こうみんかん、ドア、マップの説明パネルは `hokapan.html` 下部の `panels` を編集する
    - おみせの遷移先は `.shop-button` の `href` を編集する
- Galleryの画像を増やしたい
    - `gallery.html` の `<div class="gallery-grid">` の中に、既存の `<figure class="gallery-item">` をコピーして追加する
    - 追加画像は公開用にWebP軽量版を作り、`*-site.webp` を参照する
- Worksの作品を増やしたい
    - 仕事・コラボ作品の画像は `images/works/client/`、自主制作作品の画像は `images/works/original/` に追加する
    - `js/works-data.js` の `worksData` 配列に、既存の作品を参考にデータを1件追加する
    - 作品画像は公開用にWebP軽量版を作り、`*-site.webp` を参照する
    - 基本項目は `title`、`year`、`category`、`kind`、`image`、`description`。画像の代替文字は `title` が自動で使われる
    - `year` を設定すると年フィルタが自動で増える。年が未確定の作品は空文字のまま `ALL YEAR` にのみ表示できる
    - 新しいカテゴリを追加するときは、同じファイルの `categoryOptions` にも選択肢を追加する
    - 新しい制作区分を追加するときは、同じファイルの `typeOptions` にも選択肢を追加する

- サイト名
    - 各HTMLファイルの `Maison Conomi`
- トップ画像
    - `images/top-site.webp`
    - 元画像は `original-images/images/top.png`
    - 表示調整は `css/style.css` の `.frame` と `.top-visual`
    - 重ねるほかほかパンダ素材は `images/top-hokahokapanda-space-cosmo-crop-site.webp`
- Aboutのプロフィール文
    - `about.html` のプロフィール部分
    - 制作物・好きなものの本文は `.about-copy-text`
    - 実績一覧と出店案内は `.about-history` と `.history-note`
    - ほかほかパンダとめんだこぬの表示は `.about-characters`
- ほかほかパンダの紹介文
    - `hokapan.html` の `hokapan-profile-list`
    - クリック時の説明文は `hokapan.html` 下部の `panels`
- ほかほかパンダページの素材
    - タイトル画像は `images/hokahokapanda/hokahokapanda-name-site.webp`
    - 固定素材は `images/hokahokapanda/fixed/`
    - ランダム表示素材は `images/hokahokapanda/random/`
- Worksの作品タイトル、年、カテゴリ、説明
    - `js/works-data.js` の `worksData`
- Worksの絞り込みカテゴリ
    - `js/works-data.js` の `categoryOptions`
- Worksの絞り込み制作区分
    - `js/works-data.js` の `typeOptions`
- Worksの作品画像
    - 仕事・コラボは `images/works/client/`、自主制作は `images/works/original/`
    - 表示用は `*-site.webp`
- Galleryの画像一覧
    - `gallery.html` の各 `gallery-item`
