# 必要な素材

## 公開用画像の前提

- サイトで読み込む画像は、基本的に軽量化した `*-site.webp` を使う
- WebPよりPNGの方が軽い小さな透明素材はPNGのまま使う
- 変換前の元画像は `original-images/` に退避してローカル保管する
- `original-images/` はGitHub Pages公開用にはアップロードしない
